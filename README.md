# Amazon Certificate Manager API

AWS Certificate Manager (ACM) handles the complexity of creating, storing, and renewing public and private SSL/TLS X.509 certificates and keys that protect your AWS websites and applications, enabling you to manage certificate lifecycles centrally.

## APIs

- **Amazon Certificate Manager API** - API for provisioning, managing, and deploying public and private SSL/TLS certificates for use with AWS services and your internal connected resources. Supports requesting certificates, describing certificate details, listing certificates, importing third-party certificates, and managing certificate tags.
